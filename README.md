# Mobile Android AI Training Course

This repository contains the source code and resources for the Mobile Android AI Training Course.

## Project Overview

JourneyAI is an Android application designed to demonstrate AI integration within a mobile environment. It features a journaling system with AI-powered insights and summaries.

## Features

- **Journaling**: Create and manage journal entries.
- **AI Integration**: (In progress) Leveraging AI models for content analysis and user assistance.
- **Modern Android Stack**: Built with Kotlin, Jetpack Compose, and modern architecture patterns.

## Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the app on an emulator or physical device.
